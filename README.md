* BUCKET.SH IS A BASH SCRIPT (IT USES BASH ARRAYS, SO IT NEEDS BASH, NOT PLAIN `sh`)
* MAKE IT EXECUTABLE ONCE, THEN RUN IT:
```
chmod +x ./bucket.sh
./bucket.sh
```
* OR RUN IT DIRECTLY WITH BASH:
```
bash ./bucket.sh
```
* -d IS DEBUG MODE, WHICH IS OPTIONAL. IT IS A DRY RUN: IT PRINTS EVERY COMMAND
  IT WOULD RUN (VIA `set -x`), UNINSTALLS NOTHING, AND DOES NOT REBOOT.
```
./bucket.sh -d
```
